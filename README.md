## 使用方式

使用 `.status` 或 `/status` 即可

## 配置项

### status__avatar_url

- 说明: bot 图像链接

- 类型: str

- 默认值: https://s2.loli.net/2025/04/20/oK6pncNCeWfIyE3.jpg

### status__backgroud_url

- 说明: 背景图片链接

- 类型: str

- 默认值: https://s2.loli.net/2025/04/20/nYN2kbs38Xc6zBj.jpg

### status__nickname

- 说明: 机器人昵称

- 类型: str

- 默认值: MuikaAI

### status__priority

- 说明: 指令优先级

- 类型: int

- 默认值: 5
